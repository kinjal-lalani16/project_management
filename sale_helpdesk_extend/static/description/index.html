Sale Helpdesk Extend

Using this module in helpdesk if we select the customer it will display the records in list who belongs to that customer and Is Sat field is set to true.A new field is added in helpdesk ticket stage if it is true then only mail functionality will run.A button is added in Helpdesk ticket form and on click of that wizard for mail will be opened and clicking send button mail will be sent.In Sales Quotation Line after setting product price if we update quantity it will not change updated price and discount in quotation line. Functionality of updating price and discount on change of quantity has been stopped.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

No configurations needed.

Step-By-Step Guidelines

When you will create Quotation/Sale Order in its line if quantity of the product will change then it will not affect the price or discount made on that product.

In this first image, ordered quantity,discount and unit price has been set for all the lines.

Now in this image when i am changing ordered quantity its discount and price remain as it is and it will not be changed.

A new field has been added in contact form which is x_is_set which will be displayed in kanban view if it is set to true.

Customer field has been made required in helpdesk ticket.

When the customer field will be changed then related list of contacts will be displayed in client customer field..

Onchange of customer field, Project field will be set automatically and also Task field will be set based on lower sequence of status, if more than 2 tasks on the same stage then task with lower seuquence number will be set.

When the customer field will be changed, list of contacts will be updated and contacts whose Is Sat is true will be fetched in the list.It will send the mail to the customers/contacts who are fetched in the list.

Note: We have not display Email address of contacts as if email is long then view will get changed.

It will send the mail if solved state is set to true in any one stage of helpdesk ticket.You can go there from configuration -> stages.

A solved button is added in helpdesk ticket view , if solved is not ticked in helpdesk ticket stage then it will throw a warning.If solved is true then it will open a wizard for mail so if usre wants to edit then mail can be sent.

We have created report PDF as well Excel as below.

If Company Type is Individual and Parent is selected on contact then Parent's Police Type's value will be set on Contact.

We have added the new field stage in form,tree and kanban view.

We have added group by stage in kanban view.

We have added a functionality that if project is created from the sale order then the project name should be like SO Number + Customer Reference.
